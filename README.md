# Popy Website

Marketing website for [Popy](https://popy.app) — a local-first productivity app for macOS.

## Pages

- `index.html` — Landing page
- `manifesto.html` — Product philosophy
- `privacy.html` — Privacy policy
- `support.html` — FAQ and contact

## Development

Static site — no build tools required. Open `index.html` in a browser or run a local server:

```bash
npx serve
```
